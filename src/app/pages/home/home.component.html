XIV Raid Hub

A website to simplify raid related workflows I find tedious with existing tools.

What this website will attempt to solve

If you currently find some part of your day to day raiding life tedious then this project wants to know about it (see contribuing)! The main focus initially will be on making it easier to compare yourself to other (better) players and see what they're doing differently.

What this website will not solve

This project isn't meant to be the go-to place for raiding guides/tips/etc.. Such things are better handled by tools maintained by a large community, such as the balance. Basically, if it's not something that can be mostly automated across patches and content changes then this isn't the place for it.

This project will also avoid "reinventing the wheel" as much as possible. If some other project exists that handles a problem, then I would rather build a workflow in this project that utilizes it (with proper credit given), or simply link to it.
