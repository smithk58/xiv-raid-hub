Tools

All of the tools I currently leverage are listed below. Please support them!

  • FFLogs - I use their API to get information on fights and combat logs for players.
  • XIV API - I use their API to get information on players and servers.
  • XIV Analysis - For now I only add a convenient quick link to their analysis on reports, but in the future will explore the possibility of retrieving/saving bits of the analysis for historical comparison.

Browser Compatibility

This project will attempt to support any modern browser, but will most likely only be tested in latest versions of Firefox, Chrome, and occasionally Edge.

Contributing

Any suggestions that would make your day to day raiding easier are welcomed, just open an issue on GitHub. Code contributions are also welcomed, but you should run your idea by me in an issue on GitHub first.

Please keep in mind I have very little interest in implementing functionality that's solved by or better handled by existing tools. The community as a whole will be far better off if the existing tool is improved instead.
