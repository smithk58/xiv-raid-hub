Intro

Welcome to the XIV Raid Hub! This is an application that's meant to help simplify a lot of the extra work I find myself doing between raid days. I believe we already have quite a few useful tools for analyzing reports in various ways, but they're completely disjointed. My hope is this project can offer an easier way for myself and others to analyze themselves, their friends, and their static members over time.

It's important to understand that I do not intend for this project to compete with any other tools that I'm aware of. I have no interest in reinventing the wheel, so to speak. The bulk of the "real work" will be coming from me utilizing other tools. This application will be kept very lightweight, so please support the tools I'm leveraging instead.

Tools

All of the tools I currently leverage are listed below. Please support them!

  • FFLogs - I use their API to get information on fights and combat logs for players.
  • XIV API - I use their API to get information on players and servers.
  • XIV Analysis - For now I only add a convenient quick link to their analysis on reports, but in the future will explore the possibility of retrieving/saving bits of the analysis for historical comparison. The project is MIT, so odds are I will have to host it myself unless they add/allow a public API.
  • XIV Raid DPS Calculator - For now I only add a convenient quick link to their analysis next to reports, but in the future will explore the possibility of retrieving/saving bits of the analysis for historical comparison (the project doesn't have a license yet, so it might stay as a simple quick link to their website forever).

Browser Compatibility

This project will attempt to support any modern browser, but will most likely only be tested in latest versions of Firefox, Chrome, and occasionally Edge.

Contributing

Any suggestions that would make your day to day raiding easier are welcomed, just open an issue on GitHub. Code contributions are also welcomed, but you should run your idea by me in an issue on GitHub first.

Please keep in mind I have very little interest in implementing functionality that's solved by or better handled by existing tools. The community as a whole will be far better off if the existing tool is improved instead.
